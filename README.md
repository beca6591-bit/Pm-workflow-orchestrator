PM Workflow Orchestrator 是一个产品经理工作流编排器。它不试图替代所有专业 PM Skill，而是负责把需求发现、业务建模、PRD、原型、流程图、终稿和版本迭代串成一个有确认节点的流程。它可以单独使用，也可以和 PRD 写作、数据流分析、行业需求分析、评审类等专业 PM Skill 组合使用，作为整个 PM 工作流的流程层。

怎么用 PM Workflow Orchestrator

把 pm-workflow-orchestrator 安装到 skills 目录后，在对话里直接点名使用它即可。

使用 pm-workflow-orchestrator，帮我梳理一个新需求；

或者：

用 PM Workflow Orchestrator，基于这张截图和这段描述，帮我一步步产出 PRD。
